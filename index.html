Sphinx Prompt

.. contents:: Table of contents

Inititlise

In conf.py add extensions += ['sphinx-prompt'].

Syntax

.. code::

.. prompt:: [<language> [<prompts> [<modifier>]]]

   <statements>

Language


Supported language:

- ``text`` (no pigments, default)
- ``bash``
- ``python``
- ``scala``

Prompt(s)

If modifier is auto, a coma separated list of prompts to find in the statements.

Else the prompt to add on each statements, for Python and Bash language the end \ is supported.

Default to $,#,>>>.

Examples

.. code::

.. prompt:: bash $ noauto

    cd <folder>
    cp <src> \
        <dst>
    cd -

=>

</pre>

    # exit
    $
